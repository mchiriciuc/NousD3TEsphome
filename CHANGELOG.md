# Changelog

All notable changes to the NOUS D3T ESPHome firmware.

## [2.0.0] - 2024-XX-XX

### Added
- Dashboard Import Support
- Package-Based Architecture
- Made for ESPHome Compliance
- Comprehensive Documentation (README, CALIBRATION, MIGRATION, QUICKSTART)
- Example Configurations
- Example secrets.yaml template

### Changed
- BREAKING: WiFi config moved to device-specific files
- BREAKING: Project name: NOUS.D3T -> nous.d3t-25a
- Restructured for package imports
- Version: 2.0.0

### Improved
- Multi-device management simplified
- GitHub package distribution
- Configuration files: 300+ lines -> ~15 lines per device

## [1.5.1] - 2024-XX-XX

### Fixed
- BL0942 register configuration issues
- Improved power measurement accuracy
- Simplified component implementation

## [1.5.0] - 2024-XX-XX

### Added
- Initial public release
- BL0942 energy monitoring
- Temperature/Voltage/Current protections
- Configurable thresholds
- Web interface
- Home Assistant integration
